# Thumbnail Style Presets Quick Reference

## Tech Style
- Blue and purple gradients
- Circuit patterns, glowing accents
- Modern, professional
- **Best for:** Tech tutorials, AI/ML content, software demos

## Tutorial Style
- Bright, educational colors
- Step-by-step visual elements
- Friendly, approachable
- **Best for:** How-to videos, courses, educational content

## Gaming Style
- Vibrant, neon colors
- Dynamic composition
- Energetic, exciting
- **Best for:** Gaming content, esports, entertainment

## Business Style
- Blue and gray tones
- Clean, minimal design
- Executive, trustworthy
- **Best for:** Business tips, corporate videos, professional advice

## Creative Style
- Bold, artistic colors
- Abstract elements
- Unique, memorable
- **Best for:** Creative projects, art videos, design content

## News Style
- Red and blue accents
- Bold headlines
- Urgent, important
- **Best for:** News, updates, announcements, breaking content

## Review Style
- Product showcase
- Rating stars/scores
- Comparison elements
- **Best for:** Product reviews, comparisons, recommendations

---

## Usage Pattern

```bash
bash scripts/thumbnail.sh -s <style> "Video Title"
```

## Example Commands

```bash
# Tech thumbnail
bash scripts/thumbnail.sh -s tech "Build Your First AI App"

# Tutorial thumbnail  
bash scripts/thumbnail.sh -s tutorial "Excel Formulas Masterclass"

# Gaming thumbnail
bash scripts/thumbnail.sh -s gaming "Speedrun World Record"

# Business thumbnail
bash scripts/thumbnail.sh -s business "Startup Funding Secrets"

# Creative thumbnail
bash scripts/thumbnail.sh -s creative "Art Process Timelapse"

# News thumbnail
bash scripts/thumbnail.sh -s news "AI Breakthrough 2026"

# Review thumbnail
bash scripts/thumbnail.sh -s review "Best Laptops 2026"
```
