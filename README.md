# CometAPI Thumbnail Generator

Generate YouTube/video thumbnails quickly using AI.

## When to Use

Use this skill when the user asks to:
- Create YouTube thumbnails
- Generate video thumbnails or cover images
- Create quick image assets for video content
- Generate thumbnails with specific style presets (tech, tutorial, gaming, business, creative, news, review)

**Example prompts:**
- "Create a thumbnail for my tech tutorial video"
- "Generate a YouTube thumbnail for a gaming video"
- "Make a thumbnail image for my business review video"
